# quizzically-app
 A Quiz App which includes various categories. 

 Key Components:

 * Home Page
 * Quizzes by Category
 * Rules Page while attempting Quiz
 * MCQ based Quiz Question Page
 * Result Page with Answers & Score
 * Search Quiz
 * Quit Quiz
 * Timer for Quiz Questions
 * Authentication 
    - Login
    - SignUp
    - Logout
    - Forgot Password
